# holocure-auto-fishing-bots

## How to use
1. Open your holocure game 
2. Setting your keyboard key bind with default value 
    - RIGHT: arrow right 
    - LEFT: arrow left
    - UP: arrow up
    - DOWN: arrow down
    - CONFIRM: Z
3. Go to holohouse and stand in front of the pond
4. Run holocure_fishing_bot.exe  

## How to stop
1. Close the .exe
